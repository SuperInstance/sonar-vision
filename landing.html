SonarEncoder

ViT-adapted for acoustic data. 4 input channels: raw intensity, depth gradient, depth-normalized, and accumulated returns.

🔄

Streaming GCT

Geometric Context Transformer with causal temporal attention, paged KV cache, 3D RoPE on (depth, bearing, time), grouped-query attention.

🌊

UnderwaterColorHead

Models blue-green shift and red absorption with depth. Beer-Lambert attenuation per RGB channel.

⚡

WaterColumnModel

Mackenzie sound speed, Francois-Garrison absorption, Beer-Lambert light attenuation. Real underwater physics.

🎯

DepthWeightedLoss

Automatic ground truth selection. Cameras closest to sonar detections get exponential weight. No human annotation needed.

💾

KV Cache

Paged cache with init → update → trim → reset lifecycle. Sliding window bounds memory. Stream one sweep at a time.

// self-supervision

Zero Labels. Pure Physics.

Mount 4 GoPro cameras on a downrigger at 5m intervals. Let the sonar run. The system trains itself from the alignment of acoustic and optical sensors.

from sonar_vision.pipeline import SonarVision # Create model model = SonarVision(max_depth=200, bearing_bins=128, embed_dim=1024) # Training — cameras provide automatic ground truth output = model(sonar_intensity, camera_frames, camera_depths, sonar_detections) loss = output["loss"] # w = exp(-|d_cam - d_sonar|²/2σ²) # Streaming inference — real-time on Jetson cache = None for sweep in sonar_stream: frame, depth, cache = model.generate_stream(sweep, cache=cache)
