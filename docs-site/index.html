SonarVision

Deterministic underwater sonar physics engine. FLUX 9-opcode model. Bit-identical across all languages.

Get Started View on GitHub

Install — One Command Per Language

Rust

cargo add sonar-vision-physics

Pure Rust, 0 deps, serde optional

Python

pip install sonar-vision-physics

Stdlib only. Includes CLI + WebSocket server

TypeScript

npm install @superinstance/sonar-vision-tool

Zod schema. Works in any JS runtime

Features

≡ Deterministic

Same inputs → same outputs. Bit-identical across Rust, Python, and TS runtimes.

🌊 9-Opcode Physics

Francois-Garrison absorption, Rayleigh scattering, Mackenzie sound speed, Snell refraction.

🏡 Seasonal Thermocline

Summer peak at 15m (22°C). Winter peak at 40m (8°C). Gaussian gradient.

🏔 Sediment Types

Mud, sand, gravel, rock, seagrass — each with unique reflectivity profiles.

📡 WebSocket Streaming

Live dive simulation at configurable rate. Stdlib server, zero dependencies.

📊 Live Dashboard

Waterfall visualization, depth profiles, real-time metrics. Works in any browser.

Usage

# Python — single ping
from sonar_vision_physics import compute_physics

ping = compute_physics(depth=25.0, chlorophyll=3.0)
print(ping['temperature'])     # 8.2°C
print(ping['sound_speed'])      # 1485.0 m/s
print(ping['visibility'])       # 25.0 m
# Rust
use sonar_vision_physics::{Ping, Season, Sediment};

let ping = Ping::new(15.0, 4.0, Season::Summer, Sediment::Sand);
assert_eq!(ping.water_type_name(), "Oceanic Type II");
assert!((ping.temperature - 22.0).abs() < 0.1);
// TypeScript
import { SonarVisionTool } from '@superinstance/sonar-vision-tool';

const tool = new SonarVisionTool();
const result = await tool.execute({ action: 'physics', depth: 15 });
console.log(result.temperature); // 22
